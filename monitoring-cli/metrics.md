## 表示内容

### GPU

- グラフィックボード名
- 処理量（%）

### VRAM

- 使用量
- 容量

### CPU

- CPU 名
- コア数分のプログレスバー
- 処理量（%）

### Memory

- メモリ使用量
- メモリ容量

### プログレスバー表示

- パーセントは、小数点以下は切り上げ。
- "⣿" は 2 パーセント分。
- "⣇" は 1 パーセント分。奇数の時に使用量の最後に一度だけ使用する。
- "⣀" は空き容量を埋める。
- パーセントは "nnn %" で表示。"nnn" は、スペースパディングで、数値の右寄せ。

### 文字

- すべて大文字
